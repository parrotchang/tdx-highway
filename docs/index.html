🚀 部署步驟（一次設定，永久使用）

  1. 前往 dash.cloudflare.com 免費註冊帳號
  2. Workers & Pages → Create Worker → 貼上 worker/index.js → Deploy
  3. Worker → Settings → Variables → 新增：
    TDX_CLIENT_ID = 你的 Client ID
    TDX_CLIENT_SECRET = 你的 Client Secret
  4. 複製 Worker URL（如 https://tdx-highway.xxx.workers.dev）
  5. 開啟 docs/config.js，填入 WORKER_URL
  6. 將整個 docs/ 資料夾推送到 GitHub，開啟 GitHub Pages 即完成
